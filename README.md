## Corvid Agent & AlgoChat

I created [corvid-agent](https://github.com/CorvidLabs/corvid-agent), a decentralized AI agent platform with on-chain identity and encrypted messaging via Algorand. The agent ships its own work autonomously: [54 repositories](https://github.com/corvid-agent) and counting, including 20 web apps, 13 TypeScript libraries, and 7 macOS apps.

| Project | Description |
